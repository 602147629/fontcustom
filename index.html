Font Custom

Generates icon webfonts and a corresponding CSS file from a collection of vector images.

Generates fonts automatically

Images are automatically packed together into a font file. No messing with outdated font software or confusing scripting APIs.

Cross-browser

Font Custom generates font files for every browser. Whether its ttf, woff, eot, or svg, you're covered.

Outputs CSS file

No need to worry about what character each image is mapped too. The CSS file is mapped by filename; if you know the glyph's filename, you know the CSS class to use it.

Screen reader compatible

Some glyph fonts use letters or alphanumeric characters, which will be read out by screen readers. Font Custom fonts are prepared in such a way that this won't happen.

Use any vector images

Rather than being restricted to a pre-defined set of images, use whatever vector images you would like. It doesn't matter whether they're from Glyphicons or The Noun Project.

Infinite scalability

Because they're stored in the font as vectors, each glyph is infinitely scalable. No more @2x.

Style with CSS

Modify colors, add shadows, rotate, flip—modify and style icons however you like with just a little CSS.

Smaller glyph fonts

Fonts like Pictos and Font Awesome are great, but the number of included icons bloat filesizes and can be overkill. Font Custom only includes the icons you choose.

Updates instantly

Add an icon? Remove an icon? Fonts and the CSS file are regenerated instantly as long as Font Custom is watching.

Usage

Using Font Custom is incredibly easy. Just create a directory for your svg and eps files. Then, on the command line, run: 

fontcustom watch ~/path/to/vectors

Font Custom will create the font files and accompanying CSS file. It will also continue to run and watch the folder for changes, so you don't have to manually re-run it each time you update the image set.

Installation

Ready to get started with Font Custom? We've made sure that installation is incredibly easy. You'll need Ruby and RubyGems before you get started.

To install on OS X using Homebrew simply run:

brew install fontforge
gem install fontcustom

On Linux? No problem*:

sudo apt-get install fontforge
gem install fontcustom

* We haven't tested on Linux. Theoretically, it should work but please let us know if it doesn't.
